git 常用命令底层原理

git clone 具体干了些啥？

git push 具体干了些啥？

git log

git status

当前工作目录和 HEAD 做对比

git diff

显示当前工作目录和 HEAD 的文件差异，--cached 选项用来比较暂存区的文件 和 HEAD指向的文件。

git mv 和 git rm
